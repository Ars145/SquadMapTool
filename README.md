# Squad Map Boundary Editor

Утилита для разметки границ карт игры Squad. Загружаете изображение карты, рисуете полигон лассо-инструментом — за пределами полигона накладывается диагональная штриховка и красное свечение по границе. Экспорт в PNG в исходном разрешении.
Можно наложить картинку шаблон, чтобы ориентироваться где границы находятся. В рендере шаблон не участвует.

# https://ars145.github.io/SquadMapTool/




## Возможности

- Загрузка PNG/JPG через drag-and-drop или кнопку
- Рисование полигона по точкам, автозамыкание у стартовой точки
- Перетаскивание вершин после замыкания
- Зум колесом мыши (с центровкой на курсоре), панорамирование средней кнопкой или Space + ЛКМ
- Экспорт в PNG в полном разрешении исходного изображения

## Технологии

React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui, HTML5 Canvas, Express 5.

## Запуск

```
npm install
npm run dev
```

Сервер поднимается на порту 5000 (переопределяется переменной `PORT`).

## Сборка

```
npm run build
npm start
```
